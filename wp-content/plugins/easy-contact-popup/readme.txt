== Description ==
**Easy Contact PopUP** is the quickest and easy way to display popup contact from in your website. If you have a magazine or blogging site then it's a highly recommend plugin for your website/blog.

= Plugin Features =
* Live customizer
* Multi contact forms supported
* Easily customizable

= Supported Contact Forms =
* Contact form 7
* WPForms
* Gravity forms
* Ninja forms
* weForms

Easy Contact Popup is powered by Brainmade.co

== Installation ==
= Easy Contact PopUP is easy to install and configure. =

1. Upload the `Easy Contact PopUP` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Visit the Easy Contact PopUP options page.
